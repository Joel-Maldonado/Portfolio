Vision-Based Fight Detection

A series of deep learning models built using TensorFlow for detecting violent scenes in surveillance camera footage. This project was completed as part of the Programming for Social Good internship at Saturday Academy, which aims to use technology to create positive social impact. The model takes in a video and rates how violent it thinks it is as a percentage, along with a bounding box over the presumed fight and the people in it. By developing a model that can quickly detect violent scenes in surveillance footage, this project has the potential to improve public safety and help law enforcement respond to incidents more effectively. This project was presented as part of the ASE Saturday Academy Symposium 2022.

NeuralNetPy

NeuralNetPy is a neural network library built using Numpy and raw Python. Inspired by PyTorch, it allows you to create, train, and save models with ease, and includes common layers, activations, losses, and optimizers. The library also provides helpful utility functions for data preprocessing, training and testing the models, and saving and loading models. This project was designed for educational purposes, making it a great tool for those interested in deep learning to learn the fundamentals of building neural networks.
